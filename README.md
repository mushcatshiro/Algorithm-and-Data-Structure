## a guide to python function time complexity

| operation | Average Case | Amortized Worst Case |
|-----------|--------------|----------------------|
| copy | O(n) | O(n) |
| append | O(1) | O(1) |
| pop (last) | O(1) | O(1) |
| pop (intermediate) | O(k) | O(k)
| x in s | O(n) | - |
| min(s), max(s) | O(n) | - |
| len(s) | O(1) | O(1) |
